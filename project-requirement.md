Use the app_complete.py which implements the usage of unxstructured-io library to transform pdf files into the AstraDB vector database, and replace the functionnality of Unstructured-io with Docling and build 2 applications;
- one in console mode
- one with a GUI either using Gradio or Streamlit.
For the GUI version build a script which lauches the application in detached mode and displays the URL of the application on the console.
AstraDB python client sample could be used from "astra.py" sample application.
All AstraDB, Docling and Unstructured-io links are provided in the project-development skills.
Always refer to the project-development skills for the latest information and in order remain in the project requirements.
